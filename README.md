# Samir Taghiyev Portfolio Website

This is a modern responsive portfolio website built with HTML, CSS and JavaScript.

## Files

- `index.html` - main page content
- `style.css` - responsive modern design
- `script.js` - mobile menu, scroll animation and local file previews
- `assets/Samir_Taghiyev_CV_2026.pdf` - current CV file
- `assets/profile-placeholder.svg` - placeholder profile image
- `assets/certificates/` - folder for certificate files

## How to use

1. Open `index.html` in a browser.
2. Replace `assets/profile-placeholder.svg` with your real photo, or add your photo as `assets/profile.jpg` and update the image path in `index.html`.
3. Replace `assets/Samir_Taghiyev_CV_2026.pdf` with your latest CV if needed.
4. Put certificate images/PDFs inside `assets/certificates`.
5. Upload the folder to GitHub Pages, Netlify, Vercel or any static hosting provider.

## Important note about upload buttons

The upload buttons preview files locally in the browser. A static HTML website cannot permanently save uploaded files unless it is connected to a backend/server. For publishing, manually place your final photo, CV and certificate files in the `assets` folder.
